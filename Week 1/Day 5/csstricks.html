CSS-Tricks

The Shapes of CSS

All of the below use only a single HTML element. Any kind of CSS goes, as long as it's supported in at least one browser.

Square

.square {
height: 150px;
width: 150px;
background-color: red;
}

Circle

.circle {
height: 150px;
width: 150px;
background-color: red;
border-radius: 50%;
}
